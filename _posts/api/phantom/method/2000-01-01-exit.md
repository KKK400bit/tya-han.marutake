`phantom.exit(returnValue)` {void}

Exits the program with the specified return value. If no return value is specified, it is set to `0`.

## Examples

```javascript
if (somethingIsWrong) {
  throw new Error('myProject.something failed to locate whatever.etc');
  phantom.exit(1);
} else {
  phantom.exit(0);
}
```
