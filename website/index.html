Full web stack
No browser required

PhantomJS is a headless WebKit with JavaScript API. It has fast and native support for various web standards: DOM handling, CSS selector, JSON, Canvas, and SVG.
PhantomJS is created by Ariya Hidayat.

Download v1.4 Get started 

Simple Javascript example

console.log('Loading a web page');
var page = new WebPage();
var url = "http://www.phantomjs.org/";
page.open(url, function (status) {
    //Page is loaded!
    phantom.exit();
});

PhantomJS is an optimal solution for

Headless Website Testing

PhantomJS works with testing frameworks such as Jasmine, QUnit or CasperJS.

Site Scraping

Access and manipulate webpages with the standard DOM API, or with usual libraries like jQuery.

SVG Rendering

PhantomJS is built with the popular rendering engine Webkit and can render SVG graphics.

Network Monitoring

PhantomJS can generate HAR files to for page loading speed monitoring.
